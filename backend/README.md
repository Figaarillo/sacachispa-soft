# Sacachispa soft: Proyecto Ecomerce

API SERVER REST para el proyecto de la mataeria Testing - UTN FRVM

## Estructura del proyecto

```bash
.
├── cmd
│   └── api
├── doc
├── internal
│   ├── config
│   ├── entity
│   ├── exeption
│   ├── handler
│   ├── mock
│   ├── repository
│   ├── router
│   ├── setup
│   ├── test

│   └── util
└── migrations
```

## Comandos

`Levantar servidores:`

```bash
make docker.run
```

`Detener servidores:`

```bash
make docker.stop
```
